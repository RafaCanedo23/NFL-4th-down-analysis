# My TOG Project

This repository contains the code for a machine learning project that predicts the outcome of 4th down plays in American football.

## Project Structure
- config.py
- EDA
    - EDA_problem.ipynb
    - EDA_stats.ipynb
- data_preparation.py
- model_training.py
- neural_network_training.py
- data/fourth_down_data.csv
